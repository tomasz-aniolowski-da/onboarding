# Admin Checklist — Onboarding a New Employee

Steps to complete when a new person joins Datavid.

## Microsoft Account

- [ ] Create Microsoft account (gives access to email, calendar, and all Datavid apps)
- [ ] Inform them to change password and set up MFA

## Clockify

- [ ] Send Clockify invite
- [ ] Assign to correct project(s)
- [ ] Send invoice template and instructions (invoices@datavid.com)

## GitHub

- [ ] Invite to [datavid](https://github.com/datavid) organization
- [ ] Grant access to relevant repositories for the person's project
- [ ] Instruct them to create a new GitHub account using their Datavid email (recommended)
