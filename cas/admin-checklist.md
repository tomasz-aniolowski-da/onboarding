# Admin Checklist — CAS

Steps to complete when onboarding a new team member to CAS.

## CAS Account

- [ ] Request CAS account for the new employee
- [ ] Inform them to set password and MFA once they receive the email

## Slack

- [ ] Ensure access to [cas-acs.slack.com](https://cas-acs.slack.com)
- [ ] Add to relevant channels (e.g. [#atops-dev-env-requests](https://cas-acs.slack.com/archives/C06AXEEE4QG))

## Atlassian (Jira & Confluence)

- [ ] Ensure access to https://acs-cas.atlassian.net

## AWS WorkSpaces

- [ ] Request AWS WorkSpaces access

## AVD (Azure Virtual Desktop)

- [ ] Request AVD access if needed (required for BPM links)

## Nucleus

- [ ] Request Nucleus access (contact: **Courtney Swecker** — C_Swecker@acs.org)

## GitHub

- [ ] Grant access to [cas-claude-skills](https://github.com/datavid/cas-claude-skills) repository
